可將第2章所有範例編譯的指令批次檔。
　請依照使用的場合，依此內容做為參考，像在Linux中改寫成shell script就行了。
　請將CD-ROM中c1目錄複製到硬碟中再執行，它會幫您用javac指令進行編譯。

Penguin

操作方法
在照片範圍中用滑鼠按一下即會在按下的位置顯示一個日文字"呱"的gif圖。

PenApp

上面Penguin改寫成視窗應用程式的形式的版本
請到命令列(提示字元模式)以 java 指令執行，並配合輸入顯示出結果的程式。

Moving

改寫成不是配合滑鼠，而是使用執行緒，經過一段時間就修改一次影像顯示的位置，重繪畫面的程式。
請到命令列(提示字元模式)以 java 指令執行，並配合輸入顯示出結果的程式。

Anime

以Moving.java為基礎，並加上1 2 3 4 5不斷變換的動畫效果範例程式。所謂動畫，是依序顯示不同的影像，使畫面上的影像看起來像是在動作，以做出角色的旋轉或眨眼等效果。
請到命令列(提示字元模式)以 java 指令執行，並配合輸入顯示出結果的程式。

AnimeEx

是在Anime.java的建構子裡加上MediaTracker類別的版本。是Anime.java的擴充版。
請到命令列(提示字元模式)以 java 指令執行，並配合輸入顯示出結果的程式。

AnimeExSp
Sprite
Draw

是使用Sprite類別寫出的AnimeEx.java。將影像讀入池內的方法、將影像放在平面上的方法、控制平面上動畫功能的方法、在平面上放上文字的方法、移動平面顯示位置的方法，以及畫出Sprite的方法等，請參考書中內容。
請到命令列(提示字元模式)以 java 指令執行，並配合輸入顯示出結果的程式。
