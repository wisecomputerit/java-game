第1章

make.bat

可將第1章所有範例編譯的指令批次檔。
　請依照使用的場合，依此內容做為參考，像在Linux中改寫成shell script就行了。
　請將CD-ROM中c1目錄複製到硬碟中再執行，它會幫您用javac指令進行編譯。

Sort

是一支將命令列引數傳入之數值加以排序後顯示出來的程式，請實際編譯並執行之 。
請到命令列(提示字元模式)以 java 指令執行，並配合輸入顯示出結果的程式。

DialogSort

是改用交談的形式請求使用者輸入要排序的值，並將這些數值排序加以顯示的程式範例。
請到命令列(提示字元模式)以 java 指令執行，並配合輸入顯示出結果的程式。

Race

操作方法
